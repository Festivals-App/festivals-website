Scrinium

Let Scrinium show you the Wikipedia article, a dictionary entry, and a web search on a search engine for each technical word in your text document.
Chronisar on the Mac App Store
description faq
Analysis

With Scrinium you can open and display your text documents. The program analyzes the content, tries to filter out all technical terms from the text and displays them in a clear list. For each technical term, you can view the Wikipedia article, a dictionary entry, and a web search on a search engine, along with the document.

Flexibility

Which search engine do you use? Almost every aspect of the program can be determined by you. The search engine, the dictionary, the interface language, the presentation of the websites and many other settings.
